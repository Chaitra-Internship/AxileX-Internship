# Week 4 – Day 7 – Mini Backend Project

## Project Title
Mini Backend Project (Registration, Login, Dashboard & CRUD)

---

## Objective
To build a complete backend application with:

- User Registration
- User Login
- Dashboard
- CRUD Module
- Deployment on Render

---

## Technologies Used

- Python
- Flask
- MySQL
- mysql-connector-python
- Gunicorn
- HTML (Jinja Templates)
- Render (Hosting)
- GitHub

---

##  Features Implemented

### 1. Registration
- User can register
- Data stored in MySQL database

### 2. Login
- User validation from database
- Redirect to dashboard on success
- Show error on failure

### 3. Dashboard
- Displays student records
- Accessible only after login

### 4. CRUD Module
- Add record
- View record
- Edit record
- Delete record

### 5. Logout
- Clears session
- Redirects to login page
