# Job Application Tracker

A full-stack web application that helps users manage and track their job applications.  
Users can register, log in, add job applications, update job status, and delete records.

---

## Live Deployment

Frontend (Netlify)  
https://job-application-tracker-mini-project.netlify.app

Backend API (Render)  
https://axilex-internship-1.onrender.com/api

---

## Features

- User Registration
- User Login & Logout
- Session-based Authentication
- Add Job Applications
- Edit Job Applications
- Delete Job Applications
- Dashboard displaying all applications
- Status badges (Applied, Interview, Rejected, Offer)
- Responsive UI using Bootstrap
